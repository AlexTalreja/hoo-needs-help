# Pages

Page-level components for different routes in the application.

## Structure

- **Student**: Student-facing pages (chat, course dashboard)
- **TA**: TA workspace pages (review queue, knowledge base)
- **Instructor**: Instructor pages (course management, analytics)
- **Auth**: Login/signup pages

## Guidelines

- Each page should be a top-level route
- Use React Router for navigation
- Handle loading and error states
- Implement proper authentication checks
