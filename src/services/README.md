# Services

Service layer for API calls and external integrations.

## Files

- `supabase.ts`: Supabase client configuration
- `auth.service.ts`: Authentication services
- `rag.service.ts`: RAG query services
- `document.service.ts`: Document processing services
- `analytics.service.ts`: Analytics data services

## Guidelines

- All API calls should go through services
- Handle errors consistently
- Use TypeScript for type safety
- Add proper error messages
